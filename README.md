# DB Portal

DDBJ の登録・検索サービスへの統合ポータル。

横断検索と登録ナビゲーションを提供し、分散した DDBJ のデータベース・サービスへの統一的な入口として機能する。

## ドキュメント

- [overview.md](docs/overview.md) - プロジェクト概要・スコープ・技術スタック
- [search.md](docs/search.md) - 検索仕様（横断検索・DB 指定検索・検索エンジン対応）
- [submit.md](docs/submit.md) - 登録ナビゲーション仕様（ユースケース・ウィザード・ゴール）
- [development.md](docs/development.md) - 開発環境・テスト
- [deployment.md](docs/deployment.md) - デプロイ・環境変数
- [keycloak-setup.md](docs/keycloak-setup.md) - Keycloak クライアント設定
